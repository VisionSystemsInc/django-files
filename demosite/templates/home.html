django-files

Introduction

About

The django-files app is a small attachments framework inspired by bartTC's django-attachments and the django.contrib.comments framework.

  • Basic usage
  • View attachment details
  • Upload attachments
  • Download attachments

Features

  • Attachment relations using the ContentType framework. This means that attachments can be related to any model in your application with no extra work.
  • Database storage backend. Save your files directly in your attachment table. Currently only PostgreSQL and SQLite are supported.


    This feature should be used with caution if you have lots of attachments. Storing files in the filesystem (after all, that's what filesystems do best) are much more efficient performance wise.

  • Template tags system adapted from the comments framework. If you know how to use django.contrib.comments, managing attachments with django-files should feel right at home!
